# Tesla Stock Price Prediction (Deep Learning)

This project predicts Tesla stock prices using SimpleRNN and LSTM models.

## Features
- Data Cleaning & Preprocessing
- EDA & Visualization
- Deep Learning Models (RNN & LSTM)
- Model Comparison
- Future Prediction (1, 5, 10 days)
- Streamlit Web App

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
